# üî≠ GWFlows Project Overview

A machine learning toolkit to train and make inference on lensed gravitational waves. This pipeline helps generate posterior distribution of $\theta$ parameters of a lensed GW event within fraction of seconds, whereas the traditional MCMC would take hours to days.

# üñ•Ô∏è Command Line Interface (CLI)
`generate_dataset_gwflows` : --num-samples [int] --snr-prior [str : powerlaw,uniform]
`nf_train_gwflows` : --config-path [str: name of the .yaml file which provides configuration for normalizing flow model] --data-path [str: name of dataset file]
`nf_inference_gwflows` --config-path [str: name of the .yaml file which provides configuration for instantiating the normalizing flow model which is then loaded onto a pre-saved model] --data-path [str: name of dataset file]
